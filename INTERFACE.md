# Web modding (schema 1)

- **`bakugo-rising.css`** — `https://*/*` + `http://*/*`: mist, links, scrollbars, `::selection`, placeholders; `:root` tokens for Opera/GX.
- **`sites-01` … `sites-06`** — Per-host tweaks (`manifest.json` → `page_styles`).
- **`bakugo-opera.css`** — `https://*.opera.com/*`
- **`bakugo-gx-surfaces.css`** — `gx.games`, `gx.me`, etc.

Shaders, sounds, and wallpaper are separate from `page_styles`.
